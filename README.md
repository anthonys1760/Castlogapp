# CastLog

**Your Fishing Journal, Finally Working For You.**

CastLog is a fishing journal landing page built to capture early access waitlist signups. It positions CastLog as the privacy-first alternative to Fishbrain, which was acquired by Aspira (a government fishing license company) in November 2023.

## Live Site

[https://anthonys1760.github.io/Castlogapp/](https://anthonys1760.github.io/Castlogapp/)

## What's On The Page

- **Hero** — Waitlist signup with animated iPhone mockups showcasing three app screens
- **Problem Section** — Four pain points with existing fishing apps (spot burning, paywalls, social clutter, slow logging)
- **Feature Sections** — Deep dives into the Journal, My Spots (private by default), and AI Fish ID screens
- **Betrayal Timeline** — Fishbrain's acquisition history from SoftBank funding to Aspira buyout
- **Comparison Table** — CastLog vs. Fishbrain feature-by-feature
- **Final CTA** — Second waitlist signup with early access perks

## Tech

Single `index.html` file — no build step, no dependencies, no framework.

- Vanilla HTML/CSS/JS
- Google Fonts: Playfair Display, DM Sans, DM Mono
- Hosted via GitHub Pages

## Deployment

Hosted on GitHub Pages from the `main` branch root. To update, push changes to `main`.
